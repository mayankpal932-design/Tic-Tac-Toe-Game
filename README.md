# Tic Tac Toe Game

This is a simple and interactive **Tic Tac Toe Game** built using **HTML, CSS, and JavaScript**.  
It allows two players to play the classic Tic Tac Toe game in the browser.

## Features
- Two-player gameplay
- Restart button to start a new game
- Clean and responsive UI
- Winning result display

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Player X starts the game.
2. Players take turns clicking on the grid.
3. The first player to align three symbols (row, column, or diagonal) wins.
4. Use the restart button to play again.
