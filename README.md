# ❌⭕ Tic Tac Toe Game

An interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript** with a modern UI and player turn logic.

---

## 🚀 Features
- 1 Player & 2 Player Mode
- Dynamic player turn indicator
- Win & draw detection
- Restart game functionality
- Responsive design

---

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)
